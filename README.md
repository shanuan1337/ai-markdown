# .md converter

AI-powered конвертер произвольного текста в хорошо структурированный Markdown.

## О проекте

`.md converter` — это веб-сервис, который принимает произвольный текст (заметки, логи, скопированные таблицы, фрагменты кода) и преобразует его в качественный Markdown с правильным форматированием.

Модель автоматически:

*   Определяет заголовки и выстраивает иерархию
*   Оформляет таблицы с выравниванием столбцов
*   Оборачивает код в блоки с указанием языка
*   Сохраняет смысл, улучшая читаемость

Интерфейс позволяет редактировать результат вручную с мгновенной визуализацией.

## Возможности

*   Преобразование любого текста в сырой Markdown одной кнопкой
*   Ручное редактирование Markdown с живым превью
*   Автоопределение таблиц с выравниванием
*   Тёмная тема
*   Встроенная шпаргалка по синтаксису Markdown

## Установка

### Требования

*   Python 3.8+
*   Доступ к OpenAI-совместимому API (llama.cpp, vLLM, Ollama и др.)
*   pip-пакеты: `flask`, `flask-cors`, `requests`

```bash
pip install flask flask-cors requests
```

### Запуск

```bash
python app.py
```

Открыть `http://localhost:1337`

## Переменные окружения

| Переменная | По умолчанию | Описание |
| :--- | :--- | :--- |
| `MARKDOWN_MODEL` | `MODEL_NAME` | Имя модели в API |
| `MARKDOWN_API_URL` | `API_URL` | URL API |
| `MARKDOWN_API_TOKEN` | `API_TOKEN` | Токен авторизации |

### Systemd (Linux)

```bash
sudo cp ai-markdown.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable ai-markdown
sudo systemctl start ai-markdown
```

## API

### `POST /convert`

Конвертирует текст в Markdown.

**Request:**

```json
{
  "text": "Произвольный текст для конвертации..."
}
```

**Response (200):**

```json
{
  "markdown": "# Заголовок\n\nОтформатированный текст..."
}
```

**Response (400):**

```json
{
  "error": "Пустой текст"
}
```

**Response (500):**

```json
{
  "error": "Ошибка API: ..."
}
```

## Как это работает

1.  Пользователь вставляет текст в левую панель
2.  Нажимает «Преобразовать»
3.  Сервер отправляет текст модели с системным промптом, заточенным под структурирование Markdown
4.  Модель возвращает отформатированный Markdown
5.  Markdown отображается в правой панели — сверху код, снизу визуализация
6.  Пользователь может редактировать Markdown вручную — превью обновляется мгновенно

### Системный промпт

Модель получает инструкцию:

*   Определять заголовки, списки, таблицы, блоки кода
*   Оформлять таблицы с выравниванием (`:---`, `:---:`, `---:`)
*   Оборачивать код в тройные обратные апострофы с автоопределением языка
*   Сохранять смысл полностью
*   Возвращать только Markdown, без пояснений
